What is this system?

The Medical Diagnosis System helps users enter symptoms and receive possible medical insights. Using **Artificial Intelligence (AI) and Machine Learning (ML)**, the system analyzes symptoms and provides predictions based on trained data.

Features:

  • Secure User Authentication (Register & Login)
  • AI-based Symptom Analysis
  • Data Privacy & Security
  • Real-time Predictions

How to Use:

  1. Create an account and log in.
  2. Enter your symptoms in the diagnosis form.
  3. Receive a possible medical prediction.
  4. Consult a professional for accurate medical advice.
